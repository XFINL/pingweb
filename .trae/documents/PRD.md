## 1. 产品概述

一个面向运维人员的网络诊断工具网站，提供 HTTP 和 Ping 的 IPv4/IPv6 检测能力。
- 目标用户：运维工程师、网络管理员
- 核心价值：通过简洁直观的界面，快速执行网络诊断任务

## 2. 核心功能

### 2.1 功能模块

1. **首页**: 展示四个核心工具入口卡片，跳转到对应检测页面
2. **HTTP v4 检测页**: 对指定目标进行 HTTP IPv4 连通性检测
3. **HTTP v6 检测页**: 对指定目标进行 HTTP IPv6 连通性检测
4. **Ping v4 检测页**: 对指定目标进行 Ping IPv4 检测
5. **Ping v6 检测页**: 对指定目标进行 Ping IPv6 检测

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 首页 | 工具卡片区 | 4 个玻璃态卡片，点击跳转对应检测页面 |
| HTTP v4 页 | 检测面板 | 输入目标地址，执行 HTTP v4 检测并展示结果 |
| HTTP v6 页 | 检测面板 | 输入目标地址，执行 HTTP v6 检测并展示结果 |
| Ping v4 页 | 检测面板 | 输入目标地址，执行 Ping v4 检测并展示结果 |
| Ping v6 页 | 检测面板 | 输入目标地址，执行 Ping v6 检测并展示结果 |

## 4. 用户界面设计

### 4.1 设计风格
- **背景**: 纯黑色 (#000000)
- **设计风格**: 瑞士现代主义（Swiss Modernism）—— 极简网格、无装饰、功能至上
- **卡片风格**: 玻璃液态效果（Glassmorphism）—— 高透明度、高圆角 (border-radius: 20px-24px)、毛玻璃背景模糊
- **色彩**: 以黑白灰为主，使用高亮度蓝/青作为功能色点缀
- **字体**: 使用几何无衬线字体，体现瑞士风格
- **排版**: 严格的网格系统，大量留白，左对齐为主

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 首页 | 工具卡片 | 4 个并排玻璃态卡片，每个卡片包含图标区域、标题、副标题、箭头指示器 |
| 检测页 | 结果面板 | 输入框、检测按钮、结果展示区（状态、延迟、详情） |

### 4.3 响应式
- 桌面优先：四列卡片布局
- 平板：两列卡片布局
- 移动端：单列卡片布局
